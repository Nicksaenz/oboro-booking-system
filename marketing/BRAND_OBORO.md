# Oboro Lab - referencia de marca

## Logo correcto para marketing

Usar siempre este archivo como logo principal de Oboro Lab:

`C:\Users\USER\Documents\oboro-booking-system\marketing\oboro-labs-symbol-hd.png`

Este es el simbolo naranja sobre fondo negro, sin texto, con brillo naranja.

## Uso recomendado

- Para favicon de navegador: usar solo el simbolo, centrado y con fondo transparente.
- Para app icon y PWA: usar solo el simbolo, grande y centrado.
- Para cotizaciones, volantes y piezas de venta: usar este simbolo como marca principal.
- Evitar versiones con baja resolucion, fondos claros o logos alternativos.
- Colores base de Oboro: negro, blanco y naranja encendido.
